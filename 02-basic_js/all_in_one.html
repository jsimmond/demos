Procesando eventos de tipo "onclick"

En esta pagina, incluí un bloque de JavaScript hacia el final de la pagina, esto es lo que se encuentra dentro del tag <script> cuando ven el código fuente de la pagina. Dentro de este bloque definí una función llamada processButtonClick(). Lo único que hace esta función es buscar un elemento de la pagina Web por identificador, en este caso, el elemento con identificador "demo". Este elemento corresponde a un párrafo <p>, así que tiene un atributo llamado innerHTML. Al modificar el valor de ese atributo, cambiamos la pagina sin volver a cargar la pagina.


Eso explica el código JavaScript que se encuentra en esta pagina. Falta definir cuando se ejecutara la función processButtonClick(). Una forma es asociando la función a un evento JavaScript de tipo "onclick", que se gatilla cuando el usuario hace click en algún elemento de la pagina web. En este ejemplo, asocie el evento a un botón, indicando el valor del atributo onclick.
